About

The Campfire Utilities is a versatile Discord bot designed to enhance your server's experience. It comes with a variety of features to make server management easy and engaging for your community.

Features

  • Moderation Tools (e.g., ban, softban, tempban, mute, kick, etc.)
  • Fun and Games
  • Utility Functions
  • Customization Options
  • Auto Moderation Systems
  • Powerful Command Queue System - Queue commands to run them later automatically!
  • Ballot Management - Create polls and votings for your Discord server community!
  • Powerful Embed Management - Create, build & reuse embed schemas within Discord!
  • Information-Related Commands (e.g., find a user by ID)
  • Welcomer System - Customizable message with randomization for new members!
  • AFK System with Powerful Mention Tracking - Set AFK status and track mentions while away!
